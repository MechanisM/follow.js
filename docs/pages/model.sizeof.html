Метод возвращает количество свойств объекта или массива, находящего в цепочке chain.
Если аргумент chain не был передан, то функция вернет количество верхних свойств модели.
В случае, если model(chain) возвращает не объект, то model.sizeof вернет 0.

Если указать второй аргумент deep=true, то подсчет будет включать также свойства внутренних объектов.
Если необходимо опустить первый аргумент, то его можно пропустить и указать первым значением true (т.е deep).

Данный метод можно использовать для подсчета свойств в любом JS-объекте, не имеющего отношения к модели, если передать его первым аргументом.
