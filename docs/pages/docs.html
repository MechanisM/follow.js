Библиотека состоит из следующих частей

  1. Ядро (follow.core.js) — служит конструктором моделей и обеспечивает основной функционал доступа к данным (getter/setter).
  2. Базовый функционал (follow.extend.js) — основной набор методов для работы с моделью (follow, composite, итд.)
  3. Расширение ECMA-262 (follow.ecma.js) — поддержка обратной совместимости стандарта для старых браузеров. Расширяет прототип Array методами forEach, map, filter, some, every, indexOf, а также реализует методы String.trim() и Function.bind(). Более подробно об изменениях в JavaScript 1.6 можно почитать на MDN.
  4. Поиск данных в модели (follow.select.js) — метод, использующий простые CSS-селекторы или регулярное выражение для обновления множества свойств модели, более подробно о нем написано на странице model.select.
  5. Синхронизация модели с DOM (follow.domsync.js) — часть библиотеки для автоматической синхронизации данных модели и отображения этих данных в DOM. Связь с HTML-элементами через аттрибут data-follow="chain". Для работы с DOM используется jQuery.
