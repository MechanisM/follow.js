Библиотека состоит из 5 основных частей

  1. Ядро (follow.core.js) — служит конструктором моделей и обеспечивает основной функционал доступа к данным (getter/setter).
  2. Базовый функционал (follow.extend.js) — основной набор методов для работы с моделью (follow, composite, итд.)
  3. Расширение ECMA-262 — поддержка обратной совместимости стандарта для старых браузеров. Расширяет прототип Array методами forEach, map, filter, some, every, indexOf, а также добавляет String.trim().
    Более подробно об изменениях в JavaScript 1.6 можно почитать на MDN.
  4. Маппер для коллекций (follow.mapper.js) — для дополнительного удобства выборки/обновления данных в массивах модели.
  5. Хелпер для коллекций (follow.helper.js) — также, как и маппер, облегчает работу с коллекциями в модели, перегружая стандартные методы splice, push, pop, shift и unshift, благодаря чему, автоматически срабатывают все зависимые коллбэки.

Конструктор

  • Follow([model], [storage], [clean])
  • Getters / Setters API

Методы

  • model.init([defaults])
  • model.follow(chain, callback, [mode])
  • model.unfollow([path], [callback])
  • model.toJSON([path])
  • model.clear([path], [all])
  • model.backup([name])
  • model.restore([key])
  • model.composite(chain, callback, [sensible])
  • model.dispatch(chain, [data], [filter])
  • model.merge(data)
