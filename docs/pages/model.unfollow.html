model model.unfollow( chain, [callback] )

Метод model.unfollow, как понятно из его названия, является противоположностью для model.follow. Данный метод необходим для отписки (удаления колбэков) для отслеживаемых свойств модели.

Первым аргументом (chain) можно передать несколько цепочек через пробел. В случае, если model.follow применялся с использованием регулярного выражения, то необходимо передать аналогичную строку или выражение для отписки.

Если передать ссылку на колбэк-функцию во втором необязательном аргументе (callback), то будет удален только этот обработчик.

Пример

var model = Follow('sample'); model.follow('x.y', function( value ){ alert('New value is '+ value); }); model('x.y', 10); model('x.y', 20); model.unfollow('x.y'); model('x.y', 30); alert(model); model.toJSON
