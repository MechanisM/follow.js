model Follow( [model], [storage], [clean=true] )

Конструктор Follow служит для создания новых моделей, или же для обращения к уже существующим.

  1. model название модели, по-умолчанию "default"
  2. storage объект для хранения JSON-представления модели, по-умолчанию сохраняется во внутреннем объекте
  3. clean в случае указания этого аргумента в значении true данные модели будут удалены из объекта-хранилища, использование этого флага актуально тогда, когда storage - внешний JS-объект (см. примеры ниже)

Простые примеры

var model = Follow(); var model = Follow('myModel'); var model = Follow('settings', localStorage);

Более сложный пример

var myStorage = { sample: '{"count": 0, "items": []}' } // при изменении модели, её представление (JSON) будет храниться в myStorage.sample model = Follow('sample', myStorage, true); model.follow('count', function( value ){ $('#items-count').text( value ); }); // инициируем данные модели после загрузки DOM, // чтобы созданные коллбэки обновили представление на основе модели $(function(){ model.init(); }); Getters / Setters API
