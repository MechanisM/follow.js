Во время вызова конструктора Follow сохраняется копия модели (JSON) во внутренней переменной. Таким образом, данный метод служит для восстановления первичного состояния модели в любой момент времени, после её изменения или удаления.

Пример

var storage = { data: '{"x": 1, "y": 2}' }, model = Follow('data', storage); delete storage.data; alert( model ); model.init(); alert( model ); model.follow
