model model.init( [defaults] )

Во время вызова конструктора Follow сохраняется копия модели (JSON) во внутренней переменной. Таким образом, данный метод служит для восстановления первичных данных модели в любой момент времени, после её изменения или удаления.

Данный метод будет полезен тогда, когда часть данных может храниться во внешнем хранилище, еще до создания объекта модели, таким образом, если ниже в коде существуют слушатели для этих данных, они не сработают.
В этом случае необходимо вызывать конструктор Follow с третьим аргументом clean=true и использовать метод model.init, когда это необходимо, например, после загрузки DOM или сразу после всех model.follow и model.composite.

Первый аргрумент defaults будет использован лишь в том случае, если данные модели не существовали в объекте-хранилище на момент создания модели.

Пример

var storage = { data: '{"x": 1, "y": 2}' }, model = Follow('data', storage); delete storage.data; model('z', 3); alert( model ); model.init(); alert( model ); model.follow
