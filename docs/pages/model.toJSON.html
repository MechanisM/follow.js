model model.toJSON( [chain] )

Метод model.toJSON является алиасом для model.toString, таким образом при обращении к model в контексте строки, мы автоматически получаем полное JSON-представление модели, что особенно удобно при отладке программы.

Если передать второй необязательный аргумент (chain), то можно получить JSON-представление модели для указанной цепочки.

Пример

var model = Follow('sample'); model({ pos: {x: 0, y: 0}, rand: [1,2,3,4,5] }); // получим JSON для цепочки "pos" alert( model.toJSON('pos') ); // получим JSON для цепочки "test", результатом будет "null" т.к она не существует alert( model.toJSON('test') ); // обращаемся к model в контексте строки, поэтому автоматически вызывается model.toJSON alert( model ); model.clear
